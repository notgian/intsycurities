# SokoBot

# Running the bot
To run the bot and get a solution, the sokobot.sh or sokobat.bat scripts may be used
```
# For Mac and Linux
./sokobot.bat
# For windows
./sokobot.bat
```

An additional benchmark script exists for testing how the bot performs with multiple maps.
```
# For Mac and Linux
./benchmarker.sh
# For windows. Note that this script may be broken or outdated
# as the .sh was more commonly used
./benchmarker.bat
```
